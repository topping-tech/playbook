## نحوه استفاده در GitHub

1. داخل Organization برو: `topping-tech`
2. یک Repository جدا بساز، پیشنهاد: `dashboard` یا `topping-dashboard`
3. همه فایل‌ها و فولدرهای این پکیج را Upload کن.
4. فایل اصلی داشبورد `index.html` است.
5. اگر GitHub Pages فعال شود، داشبورد با لینک مستقیم باز می‌شود.
